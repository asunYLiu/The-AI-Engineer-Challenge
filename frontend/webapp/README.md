# Next.js Frontend for The-AI-Engineer-Challenge

This project is now located at:

```
The-AI-Engineer-Challenge/frontend/webapp
```

## Getting Started

To run the development server:

```bash
cd The-AI-Engineer-Challenge/frontend/webapp
npm install  # Only needed the first time
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.
